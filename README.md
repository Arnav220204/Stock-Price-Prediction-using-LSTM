Stock Market Prediction Using Machine Learning: Tesla (TSLA) Case StudyThis repository contains a replication and substantial extension of the 2018 study "Stock Market Prediction Using Machine Learning" by Parmar et al.. While the original paper utilized Linear Regression and a unidirectional stacked Long Short-Term Memory (LSTM) network to predict raw stock closing prices, this project modernizes the pipeline to forecast equity prices for a high-volatility case study: Tesla, Inc. (TSLA).  
Project OverviewOur core implementation, referred to as "Model B v2", utilizes sequence learning on Open-High-Low-Close-Volume (OHLCV) data to forecast near-term future prices. To accurately evaluate the model's predictive power and respect the realities of market efficiency, the methodology was entirely re-engineered to address weaknesses in the original study.  
Key Enhancements & MethodologyStationary Target Variable: The model predicts the next-day log-return—a stationary, scale-invariant metric—rather than raw price levels, which eliminates the need to relearn price-level drift.  Feature Engineering: We expanded the standard five raw OHLCV inputs into twelve engineered technical indicators to provide the network with richer, analyst-level signals.  Advanced Architecture: The unidirectional stacked LSTM was replaced with a Bidirectional LSTM, doubling the context per window by accessing patterns anchored at both ends of a 60-day look-back period.  Robust Loss Function: The network is trained using Huber loss to handle Tesla's fat-tailed return distribution and mitigate the impact of extreme, single-day market shocks.  Strict Out-of-Sample Evaluation: To test true generalization, the model was evaluated on a completely unseen, six-month forward hold-out dataset (January-June 2026) rather than relying on a static historical train/test split. 
Dataset :The model is trained on 14 years of daily TSLA trading data, totaling 3,495 trading days from January 2012 to December 2025. This timeframe was selected to expose the model to three structurally distinct market regimes: the 2012-2019 early growth phase, the 2020-2021 COVID-era rally, and the 2022-2025 mature high-volatility period.  
Key ResultsVariance Explanation: On the 2026 forward hold-out, the model successfully explained 80.1% of day-to-day price variance ($R^2 = 0.801$).  Error Metrics: The model achieved a Root Mean Squared Error (RMSE) of $10.52 and a Mean Absolute Error (MAE%) of 2.12%.  Market Efficiency Observation: Directional accuracy sits at 49.5%, which is close to the 50% random baseline. This indicates that while the model successfully tracks price levels and trends, it struggles to predict reactions to unscheduled news—a pattern wholly consistent with the semi-strong Efficient Market Hypothesis. 
Conclusion: Model B v2 serves as a highly capable price-level tracking and scenario-analysis tool for TSLA. However, given its directional accuracy, it is recommended for risk management or trend visualization rather than as a standalone buy/sell signal generator.
