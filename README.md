# Stock-Price-Prediction-using-LSTM
This project forecasts Tesla (TSLA) stock prices using a Bidirectional LSTM network trained on 14 years of data. By utilizing 12 technical indicators to predict log-returns, it achieved an $R^2$ of 0.801 on a 6-month unseen 2026 hold-out. It excels at price tracking, though its 49.5% directional accuracy aligns with the Efficient Market Hypothesis.
